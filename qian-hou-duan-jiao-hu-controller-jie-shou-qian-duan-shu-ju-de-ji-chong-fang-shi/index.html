前后端交互：Controller接收前端数据的几种方式

前端可以通过form表单将数据提交到后台，而后台则是通过Controller层与前端进行交互获取前端传递过来的数据。

前端的表单可以通过使用表单的action属性、Ajax等方式进行提交

<body>
<form action="http://localhost:8080/SSM/insertUser" method="post">
    username:<input type="text" name="username" /><br>
    password:<input type="password" name="password" /> <br>
    <input type="submit" value="登录">
</form>
</body>

或

<form  method="post" onsubmit="return false" action="##" id="formtest">
    username:<input type="text" name="username" /><br>
    password:<input type="password" name="password" /> <br>
    impression:<input type="text" name="impression">
    departMent:<input type="text" name="departMent">
    <input type="button" value="登录" onclick="login()">
</form>
<script type="text/javascript" src="C:/Users/刘家康/Desktop/ssm/jquery-3.5.1.min.js"></script>
<script type="text/javascript">
function login() {
    $.ajax({
        type: "POST",
        dataType: "text",
        url: "http://localhost:8080/SSM/insertUser",//url
        data: $('#formtest').serialize(),//这个是form表单中的id   jQuery的serialize()方法通过序列化表单值
        success: function (result) {
            alert("成功")
            console.log(result);//打印服务端返回的数据(调试用)
            if (result.resultCode == 200) {
                alert("SUCCESS");
            };
        },
        error : function(s,s2,s3) {
            console.log(s)
            console.log(s2)
            console.log(s3)
            alert("异常！");
        }
    });
}
</script>

其中几个参数需要注意： dataType：预期服务器返回的数据类型，如果是返回对象那么定义成json，定义成json时必须严格是json格式，不然就无法回调SUCCESS函数，后台使用了@ResponseBody注解时（SpringMVC返回json格式的注解），前台需要定义dataType的类型，如果是String这里要定义成text; dataType的类型主要有：xml、html、script、json、jsonp、text

后台Controller则可以通过:
1.与表单中属性相同名称的参数直接获取
2.通过HttpServletRequest的getParameter()方法获取
3.通过一个bean来接收参数

@RequestMapping("/insertUser")
    @ResponseBody
    public String insertUser(Impression impression) {
//        HttpServletRequest request
//        String username = requset.getParameter("username");
//        String password = requset.getParameter("password");

        String username = impression.getUsername();
        String password = impression.getPassword();

        return username+"  "+password;
    }
